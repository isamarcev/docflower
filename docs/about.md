# DocFlow

**Версія:** 1.0.0
**Призначення:** локальний документообіг служби списання

---

Десктоп-додаток для управління документами (docx, xlsx, xls, pdf) із
підтримкою версіонування, гілкування, тегів і аудит-логу.

Усі дані зберігаються локально, без серверної частини.

---

**Стек:** Python 3.11+ · PyQt6 · SQLite

**Натисніть F1** або відкрийте `Довідка → Допомога`, щоб дізнатися як користуватися.
